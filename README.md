## material-ui

- マニュアル

[使うな。最新バージョン推奨！！](https://v4.mui.com/)

### Material-UIを使う時に入れるパッケージ

```
$ npm install @material-ui/core
```


### ※Material-UIのアイコンを使いたい時

```
$ npm install @material-ui/icons
```

### ※Material-UIを自分でスタイリングしたい時

```
$ npm install @material-ui/styles
```
